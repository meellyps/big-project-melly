# Big-Project-Melly

This big project perform automation testing on the cicle staging website **https://staging.cicle.app/**

## How to run?
Navigate to **test/java/runner** open the **runner** file and **run the code**

## Requirement

Java JDK 11

## Dependencies

Selenium 4.4.0

Cucumber 7.3.3

TestNG 7.2.3
